# Todo - A CLI Task Management Application

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## Overview

**Todo** is a minimalist and efficient command-line interface (CLI) application designed to help you manage your daily tasks and to-do lists with ease. Built with C++, it focuses on speed and simplicity, allowing you to quickly add, view, complete, and organize your tasks without leaving the terminal.

## Features

*   **Quick Task Management:** Add new tasks, mark them as complete, or remove them with simple commands.
*   **Persistent Storage:** Your tasks are saved locally, ensuring they are available across sessions.
*   **Intuitive Interface:** A clean and straightforward CLI experience.
*   **Lightweight:** Built for performance with minimal resource consumption.

## Installation

To build and install the Todo application, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/HassanIQ777/Todo.git
    cd Todo
    ```

2.  **Build with Make:**
    ```bash
    make
    ```
    
## Usage

Once installed, you can use `todo` from your terminal.

```bash
$ todo
Usage:
  todo  [PATH TO TODO FOLDER]
  -v    to show version number
  -h    to show this message
```
